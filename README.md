# Groq AI Chat

Terminal-based AI chat client written in [Rust](https://www.rust-lang.org/).  
Built with [Crossterm](https://github.com/crossterm-rs/crossterm) and [Groq API](https://groq.com/).  
You need Rust and your Groq API key in .env file as GROQ_API_KEY for building and running.

## Installation, Building and Running
```sh
$ git clone https://github.com/HasChad/groq-ai-chat
$ cd groq-ai-chat
$ cargo build run
```

Make sure you have right API key in your .env file
